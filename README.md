## 练习
### 概述

Markdown是一种轻量级的标记语言，使用纯文本格式编写文档，然后转换成html文档。跟html相比，更加注重内容。
#### 优点
- 语法简单
- 注重问题
- 易读
#### 缺点
1. 图片
2. 语法兼容
3. 细节控制
